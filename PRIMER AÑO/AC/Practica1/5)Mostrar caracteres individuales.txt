a) Escribir un programa que muestre en pantalla las letras mayúsculas (“A” a la “Z”).
Pista: Podés buscar los códigos de la “A” y la “Z” en una tabla de códigos ascii. No utilizar un vector. Usar
una sola variable de tipo db, e incrementar el valor de esa variable antes de mostrar en pantalla.
ORG 1000H
LETRA DB 040h
ORG 2000H
mov bx, offset LETRA
mov al, 1
mov ah,05Ah
loop:inc LETRA 
int 7
cmp ah,LETRA
jnz loop
int 0
END

d) Modificar el ejercicio b) que muestra los dígitos, para que cada dígito se muestre en una línea separada.
Pista: El código ASCII del carácter de nueva línea es el 10, comúnmente llamado “\n” o LF (“line feed” por
sus siglas en inglés y porque se usaba en impresoras donde había que “alimentar” una nueva línea). Entonces,
se puede imprimir el string “{c}\n”, que tiene longitud 2, donde {c} es el carácter a imprimir y \n es el carácter
de nueva línea.
ORG 1000H
LETRA DB 02Fh
LINEFEED DB 00AH
ORG 2000H
mov bx, offset LETRA
mov al, 1
mov ah,039h
loop:inc LETRA 
int 7
mov bx, offset LINEFEED
int 7
mov bx, offset LETRA
cmp ah,LETRA
jnz loop
int 0
END
